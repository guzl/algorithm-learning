# 无重复字符的最长子串
## 滑动窗口
设置两个指针，一个叫left，一个叫right，right不断的往移动，left和right之间的就是窗口，如果这个窗口里面有重复的字符，那么left向右移动，直到这个窗口内没有重复的字符；
当窗口里面没有重复的字符时，right向右移动，如此往复，直到字符串最后。
每次窗口大小变动时，需要记录窗口长度，以此找到最大的窗口长度。
- 时间复杂度
```math
O(n)
```
- 空间复杂度
```math
O(1)
```
```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        slen = len(s)
        max_len = r = l = 0
        hashtable = set()
        while r < slen:
            while s[r] in hashtable:
                hashtable.remove(s[l])
                l += 1
            hashtable.add(s[r])
            max_len = max(r-l+1,max_len)
            r += 1
        return max_len
```
## 滑动窗口变种
将滑动窗口中的左指针的移动换成一种跳跃式的，而不是一步一步的移动。
跳跃式的移动就是发现右重复的字符时，左指针应该移动到上一次出现这个字符位置和当前左指针取最大值。所以左指针时从-1开始。
```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        dic, res, left = {}, 0, -1
        for right in range(len(s)):
            if s[right] in dic:
                left = max(dic[s[right]], left)
            dic[s[right]] = right
            res = max(res, right-left)
        return res
```
