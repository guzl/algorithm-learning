# 2.两数相加
主要考你数据结构吧，会不会用。
## 关键点
1. 非空链表
2. 非负整数
3. 逆序
## 解法
### 遍历
将两个链表的每一位遍历相加即可，主要注意处理进位和长度不对等的情况
- 时间复杂度
```math
O(n)
```
- 空间复杂度
```math
O(1)
```
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        ln_ret = ListNode()
        ln_head = ln_ret
        jw = 0
        while l1 or l2 or jw:
            v1 = l1.val if l1 else 0
            v2 = l2.val if l2 else 0
            n_tmp = v1 + v2 + jw
            ln_ret.next = ListNode(int(n_tmp % 10))
            ln_ret = ln_ret.next
            jw = int(n_tmp/10)
            l1 = l1.next if l1 else None
            l2 = l2.next if l2 else None
        return ln_head.next
        
```
