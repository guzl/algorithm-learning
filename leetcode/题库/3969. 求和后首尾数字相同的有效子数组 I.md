# 3969. 求和后首尾数字相同的有效子数组 I
## 关键点
1. 无
## 解法
### 1.枚举所有子数组，找到符合条件的一个
#### 时间复杂度
```math
O(n^2)
```
#### 空间复杂度
```math
O(1)
```
#### 代码(6150ms)
```python
class Solution:
    def countValidSubarrays(self, nums: list[int], x: int) -> int:
        n = len(nums)
        ret = 0
        for i in range(n):
            b = 0
            for j in range(i,n):
                b = b + nums[j]
                last = b%10
                if last != x:
                    continue
                a = b
                while a >= 10:
                    a = a // 10
                first = a
                if first == x:
                    ret += 1
        return ret
```
