# 746. 使用最小花费爬楼梯
无。
## 关键点
1. 人站在-1的位置，-1的费用应该为0，往上走可以走到0的位置，也可以走到1的位置
## 解法
### 动态规划
走完n个楼梯的总花费all_cost[n],all_cost = min(all_cost[n-1]+cost[n-1],all_cost[n-2]+cost[n-2])
注意，all_cost[0]=0,all_cost[1]=0
- 时间复杂度
```math
O(n)
```
- 空间复杂度
```math
O(1)
```
```python
class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        n = len(cost)
        all_cost = [0]*(n+1)
        for i in range(2,n+1):
            value1 = all_cost[i-1]+cost[i-1]
            value2 = all_cost[i-2]+cost[i-2]
            all_cost[i] = min(value1,value2)
        return all_cost[n]
```
