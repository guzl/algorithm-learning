# 2144. 打折购买糖果的最小开销
## 关键点
1. 无
## 解法
### 1.贪心，数组倒序然后没三个为一组，前面两个要付钱，后面一个不付钱
#### 时间复杂度
```math
O(n*log(n))
```
#### 空间复杂度
```math
O(1)
```
#### 代码(0ms)
```python
class Solution:
    def minimumCost(self, cost: List[int]) -> int:
        ret = 0
        cost.sort(reverse=True)
        n = len(cost)
        for i in range(n//3):
            ret += (cost[i*3]+cost[i*3+1])
        for i in range(n%3):
            ret += cost[-i-1]
        return ret
```
