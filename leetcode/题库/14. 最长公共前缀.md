# 14. 最长公共前缀
## 关键点
1. 注意只是前缀。
## 解法
1. 先找到第一个和第二个的公共前缀，然后用这个公共前缀和第三个字符串比较，以此类推。
#### 代码(0ms)
```python
class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        ans = strs[0]
        for i in range(1,len(strs)):
            t = ''
            for a,b in zip(ans,strs[i]):
                if a == b:
                    t+=a
                else:
                    break
            if len(t) == 0:
                return ""
            else:
                ans = t
        return ans
```
