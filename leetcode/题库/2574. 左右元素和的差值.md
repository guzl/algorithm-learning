# 2574. 左右元素和的差值
## 关键点
1. 无
## 解法
### 1.跟着题意走就好了
#### 时间复杂度
```math
O(n^2)
```
#### 空间复杂度
```math
O(n)
```
#### 代码(4ms)
```python
class Solution:
    def leftRightDifference(self, nums: List[int]) -> List[int]:
        n = len(nums)
        answer = [0]* n
        leftSum = 0
        rightSum = sum(nums[1:]) 
        answer[0] = abs(leftSum - rightSum)
        for i in range(1,n):
            leftSum += nums[i-1]
            rightSum -= nums[i]
            answer[i] = abs(leftSum - rightSum)
        return answer
```
### 2.递推公式
$$ totle = leftSum[i] + nums[i] + rightSum[i] $$
$$ rightSum[i]  = totle - leftSum[i] - nums[i] $$
$$ |leftSum[i] - rightSum[i]| = 2* leftSum[i] + nums[i] -totle $$
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(1)
```
#### 代码(0ms)
```python
class Solution:
    def leftRightDifference(self, nums: List[int]) -> List[int]:
        total = sum(nums)
        leftSum = 0
        for i,x in enumerate(nums):
            nums[i] = abs(leftSum * 2 + x - total)
            leftSum += x
        return nums
```
