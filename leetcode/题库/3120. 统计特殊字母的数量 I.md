# 3120. 统计特殊字母的数量 I
## 关键点
1. 无
## 解法
### 1.新建两个表，一个全大写，一个全小写，遍历比较看大写和小写是否都再word中。
#### 时间复杂度
```math
O(26*log(m))
```
#### 空间复杂度
```math
O(1)
```
#### 代码(0ms)
```python
class Solution:
    def numberOfSpecialChars(self, word: str) -> int:
        ret = 0
        letter_up = [chr(ord('A')+x) for x in range(26)]
        letter_low = [chr(ord('a')+x) for x in range(26)]
        for i in range(26):
            if letter_low[i] in word and letter_up[i] in word:
                ret += 1
        return ret
```
