# 3737. 统计主要元素子数组数目 I
## 关键点
1. 无
## 解法
### 1.枚举，枚举所有子数组，对符合条件的进行计数+1，然后返回计数
### 2.再第二层循环时可以使用前一个计数的值，这样可以节省一些时间
#### 时间复杂度
```math
O(n^2)
```
#### 空间复杂度
```math
O(1)
```
#### 代码(1861ms)
```python
class Solution:
    def countMajoritySubarrays(self, nums: List[int], target: int) -> int:
        ret = 0
        n = len(nums)
        for i in range(n):
            cnt = 0
            for j in range(i+1,n+1):
                if nums[j-1] == target:
                    cnt += 1
                if cnt > (j-i)//2:
                    ret += 1
        return ret
```
