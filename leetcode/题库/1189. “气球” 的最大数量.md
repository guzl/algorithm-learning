# 1189. “气球” 的最大数量
## 关键点
1. balloon这个单词里面，ban各有一个，l和o各有两个。
## 解法
### 1.枚举
遍历整个字符串，找到balon五个字母各自的数量，求这些值的最小值(注意lo的数量要除以2)
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(n)
```
#### 代码(6150ms)
```python
class Solution:
    def maxNumberOfBalloons(self, text: str) -> int:
        str_dict = defaultdict(int)
        for s in text:
            str_dict[s]+=1
        min_ret = 10001
        if min_ret > str_dict['b']:
            min_ret = str_dict['b']
        if min_ret > str_dict['a']:
            min_ret = str_dict['a']
        if min_ret > str_dict['n']:
            min_ret = str_dict['n']
        if min_ret > (str_dict['l']//2):
            min_ret = (str_dict['l']//2)
        if min_ret > (str_dict['o']//2):
            min_ret = (str_dict['o']//2)
        return min_ret
```
