# 1.两数之和
这个是一个开篇的题目，很简单。为的是帮大家熟悉做题代码提交环境和提交过程。
## 关键点
1. 只有一个答案
2. 不能使用相同原始
## 解法
### 暴力循环
将输入数组中的元素两两相加，当等于目标值时，直接输出下标即可。
- 时间复杂度
```math
O(n^2)
```
- 空间复杂度
```math
O(1)
```
```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        for i in range(len(nums)-1):
            for j in range(i+1,len(nums)):
                if nums[i] + nums[j] == target:
                    return [i,j]
        return []
```
###哈希
只做一次遍历，将已经遍历过的数字放入一个hash表中，key为数据，value为下标，每次遍历是都查询一下hash表中有没有和当前值求和可以达到目标值的，如果有，则输出结果。
- 时间复杂度 
```math
O(n)
```
- 空间复杂度 
```math
O(n)
```
```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        hashtable = dict()
        for i in range(len(nums)):
            need = target - nums[i]
            if need in hashtable:
                return [hashtable[need],i]
            hashtable[nums[i]] = i
        return []
```
