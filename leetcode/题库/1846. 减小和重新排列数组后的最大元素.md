# 1846. 减小和重新排列数组后的最大元素
## 关键点
1. 数字只能变小，或者不变，不能变大。
## 解法
无
#### 代码(27ms)
```python
class Solution:
    def maximumElementAfterDecrementingAndRearranging(self, arr: List[int]) -> int:
        #只能减小，不能增加
        n = len(arr)
        arr.sort()
        ans = 0
        for i in range(n):
            if arr[i] >= ans+1:
                ans +=1
        return ans
```
#### 代码(3ms)
```python
class Solution:
    def maximumElementAfterDecrementingAndRearranging(self, arr: List[int]) -> int:
        #只能减小，不能增加
        arr.sort()
        ans = 0
        for x in arr:
            if x > ans:
                ans +=1
        return ans
```
通过上面的区别我们可以看出，使用迭代器比使用下标访问的速度要快得多。
