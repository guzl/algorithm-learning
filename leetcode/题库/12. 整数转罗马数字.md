# 12. 整数转罗马数字
## 关键点
1. 关于4和9的问题，我们可以直接列出来就好了。
## 解法
### 1.枚举，不断的减去字典中的值。
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(1)
```
#### 代码(0ms)
```python
class Solution:
    def intToRoman(self, num: int) -> str:
        mydict = {1000:'M',900:'CM',500:'D',400:'CD',100:'C',90:'XC',50:'L',40:'XL',10:'X',9:'IX',5:'V',4:'IV',1:'I'}
        i = 0
        ret = ''
        for k,v in mydict.items(): 
            while num>=k:
                ret += v 
                num -= k
        return ret
```
