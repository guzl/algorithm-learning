# 4.寻找两个正序数组的中位数
这个题目应该主要想考察排序的使用，很多语言都自带右排序函数，所以。。。
## 关键点
1.注意数组是空的情况
## 解法
### 1.排序->根据长度找中位数
合并两个数组，使用sort函数对合并后的数组进行排序
* 如果长度n为奇数，就返回下标为floor(n/2)的值；
* 如果长度n为偶数，就返回下标为floor(n/2)和下标为floor(n/2-1)的两个数的和的一半。
#### 时间复杂度
```math
O((m+n)*log(m+n))
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        s = (nums1 + nums2)
        s.sort()
        s_len = len(s)
        if s_len == 0:
            return 0.0
        if s_len == 1:
            return s[0]
        if s_len%2 == 1:
            return s[floor(s_len/2)]
        else:
            return (s[floor(s_len/2)] + s[floor(s_len/2) - 1])/2
```
### 2.不用系统排序，遍历找到中位数
因为两个数组已经时从小到大有序的，所以只需要知道两个数组的长度，获得中位数的位置，然后重头遍历两个数组的数字并比较大小，确定那个是中位数就可以了。
#### 时间复杂度
```math
O(m+n)
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        len1,len2 = len(nums1),len(nums2)
        all_len = len1 + len2
        if all_len == 0:
            return 0.0
        if all_len == 1:
            return nums2[0] if len1 == 0 else nums1[0]
        index = floor(all_len/2) -1
        i = j = 0
        ret1 = ret2 = 0
        while index>=0:
            index -= 1
            if i >= len1:
                ret1 = nums2[j]
                ret2 = nums2[j+1]
                j+=1
            elif j >= len2:
                ret1 = nums1[i]
                ret2 = nums1[i+1]
                i+=1
            elif i<len1 and j<len2:
                if nums1[i]<nums2[j]:
                    ret1 = nums1[i]
                    ret2 = nums1[i+1] if i+1<len1 and nums1[i+1]<nums2[j] else nums2[j]
                    i+=1
                else:
                    ret1 = nums2[j]
                    ret2 = nums2[j+1] if j+1<len2 and nums2[j+1]<nums1[i] else nums1[i]
                    j+=1   
        if all_len%2 == 1:
            return ret2
        else:
            return (ret1+ret2)/2
```
### 3.不用系统排序，使用二分法找到中位数
因为两个数组已经时从小到大有序的，所以只需要知道两个数组的长度，获得中位数的位置，然后试用二分法找确定哪个是中位数就可以了。
#### 时间复杂度
```math
O(log(m+n))
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        def getKthElement(k):
            #从两个数组的第一个元素开始
            index1,index2 = 0,0
            while True:
                #退出条件设定
                #第一个数组已经找完了，那直接返回第二个数组的第key个元素
                if index1 == m:
                    return nums2[index2 + k -1]
                #第二个数组已经找完了，那直接返回第一个数组的第key个元素
                if index2 == n:
                    return nums1[index1 + k -1]
                #当k为1时找还未查找的元素里面的最小的一个
                if k == 1:
                    return min(nums1[index1],nums2[index2])
                #生成新的二分点。
                newIndex1 = min(index1 + k//2-1,m-1)
                newIndex2 = min(index2 + k//2-1,n-1)
                if nums1[newIndex1] <= nums2[newIndex2]:
                    #因为nums1[newIndex1]是小的所有现在要查找的元素个数就要减去排除掉的个数
                    #k是个数是从1开始的，index是下标是从0开始的
                    k -= newIndex1-index1+1
                    index1 = newIndex1+1
                else:
                    #剔除nums1[newIndex1]之前的所有元素包含nums1[newIndex1]
                    k -= newIndex2 - index2+1
                    index2 = newIndex2+1
        m,n = len(nums1),len(nums2)
        totalLength = m + n
        if totalLength%2 == 1:
            return getKthElement((totalLength+1)//2)
        else:
            return (getKthElement(totalLength//2)+getKthElement(totalLength//2+1))/2
```
