# 4.寻找两个正序数组的中位数
这个题目应该主要想考察排序的使用，很多语言都自带右排序函数，所以。。。
## 关键点
1.注意数组是空的情况
## 解法
### 1.排序->根据长度找中位数
合并两个数组，使用sort函数对合并后的数组进行排序
* 如果长度n为奇数，就返回下标为floor(n/2)的值；
* 如果长度n为偶数，就返回下标为floor(n/2)和下标为floor(n/2-1)的两个数的和的一半。
#### 时间复杂度
```math
O((m+n)*log(m+n))
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        s = (nums1 + nums2)
        s.sort()
        s_len = len(s)
        if s_len == 0:
            return 0.0
        if s_len == 1:
            return s[0]
        if s_len%2 == 1:
            return s[floor(s_len/2)]
        else:
            return (s[floor(s_len/2)] + s[floor(s_len/2) - 1])/2
```
### 2.不用系统排序，遍历找到中位数
因为两个数组已经时从小到大有序的，所以只需要知道两个数组的长度，获得中位数的位置，然后重头遍历两个数组的数字并比较大小，确定那个是中位数就可以了。
#### 时间复杂度
```math
O(m+n)
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        len1,len2 = len(nums1),len(nums2)
        all_len = len1 + len2
        if all_len == 0:
            return 0.0
        if all_len == 1:
            return nums2[0] if len1 == 0 else nums1[0]
        index = floor(all_len/2) -1
        i = j = 0
        ret1 = ret2 = 0
        while index>=0:
            index -= 1
            if i >= len1:
                ret1 = nums2[j]
                ret2 = nums2[j+1]
                j+=1
            elif j >= len2:
                ret1 = nums1[i]
                ret2 = nums1[i+1]
                i+=1
            elif i<len1 and j<len2:
                if nums1[i]<nums2[j]:
                    ret1 = nums1[i]
                    ret2 = nums1[i+1] if i+1<len1 and nums1[i+1]<nums2[j] else nums2[j]
                    i+=1
                else:
                    ret1 = nums2[j]
                    ret2 = nums2[j+1] if j+1<len2 and nums2[j+1]<nums1[i] else nums1[i]
                    j+=1   
        if all_len%2 == 1:
            return ret2
        else:
            return (ret1+ret2)/2
```
