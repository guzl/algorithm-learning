# 2095. 删除链表的中间节点
## 关键点
1. 无
## 解法
### 1.两个指针，一个指针每次挪动一次，一个指针每次挪动两次
#### 时间复杂度
```math
O(n/2)
```
#### 空间复杂度
```math
O(1)
```
#### 代码(72ms)
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def deleteMiddle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        a = head
        b = head
        c = head
        if head.next == None:
            return None
        while c and c.next:
            c = c.next.next
            b = a
            a = a.next
        if b.next:
            b.next = b.next.next
        return head
```
