# 3020. 子集中元素的最大数量
## 关键点
1. 注意处理1的问题。
## 解法
1. 先指定一个值，然后顺着这个值往下走，看最最大的子集是多少。
#### 代码(105ms)
```python
class Solution:
    def maximumLength(self, nums: List[int]) -> int:
        num_dict = Counter(nums)
        #处理1的问题，因为1的不管多少次方都是1，所有如果数组中有多个1，是可以形成子集的
        #但是是奇数，[1],[1,1,1],[1,1,1,1,1]
        ret = (num_dict[1] - 1)|1
        del num_dict[1]

        for i in num_dict:
            t_num = i
            t_len = 0
            while True:
                #如果该值不在数组中，那么需要把长度减去1，
                #就是现在我们加入的最后一个元素，我们加了两个，只用加一个就好了
                if t_num not in num_dict:
                    t_len -=1
                    break
                #如果该值只有一个，那么看好最为顶尖，总数+1就好了
                if num_dict[t_num] == 1:
                    t_len += 1
                    break
                #其他的就是该值的个数>=2.
                else:
                    t_len += 2
                    t_num = t_num*t_num
            ret = max(ret,t_len)
        return ret
```
