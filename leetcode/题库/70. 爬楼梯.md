# 70.爬楼梯
这个是一个动态规划的入门题目。
## 关键点
1. 无
## 解法
### 动态规划
动态规划算法简单来说求解N可以通过求解N-1的问题来解决。
具体到这个问题，就是求n个楼梯的走法，我们假设最后一步只走一个台阶就是n，那有多少种呢？就是n-1个楼梯的走法那么多种；在假设最后一步走两个楼梯，就是走n-1和n这两个楼梯，那么有多少种呢？就是n-2个楼梯那么多种。这两个假设相加，就是n个楼梯的走法，即：
```math
f(n) = f(n-1)+f(n-2)
```
考虑边界问题，我们可以先求出:
```math
f(1)=1,f(2)=2,f(3)=f(2)+f(1).....,f(n)=f(n-1)+f(n-2)
```
- 时间复杂度
```math
O(n)
```
- 空间复杂度
```math
O(1)
```
```python
class Solution:
    def climbStairs(self, n: int) -> int:
        ret = [0]*(n+2)
        ret[0] = 0
        ret[1] = 1
        ret[2] = 2
        for i in range(3,n+1):
            ret[i] = ret[i-1]+ret[i-2]
        return ret[n]
```
