# 53. 最大子数组和
## 关键点
1. 无。
## 解法
贪心,遍历数组，如果前面的和是小于0的，子数组就从这个开始；如果是大于0的，就将这数字加上
## 优化
无
#### 代码(15ms)
```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # 贪心
        ans = pre = -inf #之前的和
        for x in nums:
            pre = x if pre < 0 else (pre+x)
            ans = pre if pre>ans else ans
        return ans
```
## 解法
动态规划
f(i)为i之前以及i的子数组最大和，那么f(i) = MAX(f(i-1)+nums[i],nums[i])
返回f[i]的最大值
## 优化
无
#### 代码(75ms)
```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        #动态规划
        #f(i) = MAX(f[i-1]+nums[i],nums[i])
        ans = 0 #最大值
        dp = defaultdict(int)
        ans = dp[0] = nums[0]
        n = len(nums)
        for i in range(1,n):
            dp[i] = max(dp[i-1]+nums[i],nums[i])
        return max(dp)
```
