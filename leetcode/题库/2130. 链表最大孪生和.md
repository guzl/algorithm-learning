# 2130. 链表最大孪生和
## 关键点
1. 无
## 解法
### 1.将链表中的值放到数组中，然后遍历
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(n)
```
#### 代码(72ms)
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def pairSum(self, head: Optional[ListNode]) -> int:
        a = head
        nums = []
        ret_max = 0
        while a:
            nums.append(a.val)
            a = a.next
        num_len = len(nums)
        for i in range(num_len//2):
            ret_max = max(nums[i] + nums[num_len-i-1],ret_max)
        return ret_max
```
