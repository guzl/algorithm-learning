# 3960.频率平衡子数组

## 关键点
1. [4,4,5,5]最长是3：[4,4,5]或[4,5,5]；
2. [5,5,5,5]最长是4
## 解法
### 1.遍历，找到最长的合乎规则的子数组
#### 时间复杂度
```math
O(n^2)
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def getLength(self, nums: List[int]) -> int:
        res = 1
        nums_len = len(nums)
        for i in range(nums_len):
            if nums_len - i <=res:
                break
            a = defaultdict(int)
            max_one = 0
            for j in range(i,nums_len):
                a[nums[j]] += 1
                if max_one < a[nums[j]]:
                    max_one = a[nums[j]]
                # 说明只有一个数字[5,5,5,5]
                if len(a.keys()) == 1:
                    res = max(res,j+1-i)
                    continue
                #最大的个数应该为偶数，才可能是其他的两倍
                if max_one%2==0:
                    #比如[4,4,5,5],这个数列是不对的
                    tmp = list(set(a.values()))
                    if len(tmp) != 2:
                        continue
                    else :
                        if tmp[0] == 2*tmp[1] or 2*tmp[0] == tmp[1]:
                            res = max(res,j+1-i)
        return res
```
