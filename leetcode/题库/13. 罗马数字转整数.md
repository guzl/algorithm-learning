# # 12. 整数转罗马数字
## 关键点
1. 关于4和9的问题，我们可以直接列出来就好了。
## 解法
### 1.枚举，先判断是否符合4和9的情况，在判断其他情况。
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(1)
```
#### 代码(0ms)
```python
class Solution:
    def romanToInt(self, s: str) -> int:
        mydict = {'M':1000,'D':500,
                 'C':100,'L':50,
                 'X':10,'IX':9,
                  'V':5,'I':1}
        mydict1 = {'CM':900,'CD':400,'XC':90,
                  'XL':40,'IX':9, 'IV':4}
        i = 0
        n = len(s)
        ret = 0
        while i < n:
            if i+2<=n and s[i:i+2] in mydict1:
                ret += mydict1[s[i:i+2]]
                i = i+2
            elif i+1<=n and s[i] in mydict:
                ret += mydict[s[i]]
                i = i+1
        return ret   
```
