# 525. 连续数组
## 关键点
1. 找最长子串的，子串要求0和1的元素个数相同。
2. 最大值肯定是偶数
3. 求前缀和sum数组，当$$ sum[j] - sum[i] = (j-i)/2 $$ j-i的最大值就是要求的值。
4. 我们让数组中的0变成-1，这样第3条的公式就可以改为$$ sum[j] - sum[i] = 0 $$
5. 那就是求$$ sum[j] = sum[i] $$ j-i最大值。
## 解法
### 1.
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(n)
```
#### 代码(99ms)
```python
class Solution:
    def countMajoritySubarrays(self, nums: List[int], target: int) -> int:
        # 把0当做-1
        nums = [x if x else -1 for x in nums]
        
        # 计算nums的前缀和
        s = list(accumulate(nums,initial=0))
        pos = {}
        ans = 0
        for i,x in enumerate(s):
            if x in pos:
                ans = max(ans,i-pos[x])
            else:
                pos[x] = i
        return ans
```

class Solution:
    def findMaxLength(self, nums: List[int]) -> int:
        pos = {0:-1}
        ans = s = 0
        for i,x in enumerate(nums):
            s += 1 if x else -1
            if s in pos:
                ans = max(ans,i-pos[s])
            else:
                pos[s] = i
        return ans
