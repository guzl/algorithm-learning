# 3968. 移动后的最大曼哈顿距离
## 关键点
1. '_'所在的位置是没有什么关系的，我们可以让它一直往又走就好了
## 解法
### 1.遍历
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(1)
```
####  代码(111ms)
```python
class Solution:
    def maxDistance(self, moves: str) -> int:
        x,y = 0,0
        n_num = 0
        for str in moves:
            if str == 'U':
                y+=1
            elif str == 'D':
                y-=1
            elif str == 'R':
                x+=1
            elif str == 'L':
                x-=1
            else:
                n_num+=1
        return abs(x)+abs(y)+n_num
```
