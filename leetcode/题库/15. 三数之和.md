# 15. 三数之和
## 关键点
1. 无
## 解法
1. 排序
2. 枚举一个数字
3. 左右指针找两个数
4. 这三个数和为0
5. 移动左右指针
#### 时间复杂度
```math
O(n^2)
```
#### 空间复杂度
```math
O(1)
```
#### 代码(6150ms)
```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        res = []
        n = len(nums)
        nums.sort()  # 排序
        for i in range(n):
            # 固定值去重，和上一个一样直接跳过
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            target = -nums[i]
            left = i + 1
            right = n - 1
            while left < right:
                s = nums[left] + nums[right]
                if s == target:
                    res.append([nums[i], nums[left], nums[right]])
                    # 左指针去重
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    # 右指针去重
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    # 同时收缩
                    left += 1
                    right -= 1
                elif s < target:
                    left += 1
                else:
                    right -= 1
        return res
```
