# 3739. 统计主要元素子数组数目 II
## 关键点
1. 这个题目和3737是一样的，只不过这个判题数据大一些。不能使用枚举的方法，不然要超时。
## 解法
1. 把nums中和target一致的改为1，不一致的改为-1
2. 求前缀和sums，问题转化为，枚举j，$$ sums[j] >sum[i],j>i, $$看符合条件的(i,j)有多少对
#### 时间复杂度
```math
O(n*log(n))
```
#### 空间复杂度
```math
O(n)
```
#### 代码(658ms)
```python
class Solution:
    def countMajoritySubarrays(self, nums: List[int], target: int) -> int:
        sl = SortedList([0]) 
        ans = s = 0
        for x in nums:
            #求前缀和
            s += 1 if x == target else -1
            # 找到比当前前缀和小的前缀和的个数，并加到结果中。
            ans += sl.bisect_left(s)
            sl.add(s)
        return ans
```

```python
class Solution:
    def countMajoritySubarrays(self, nums: List[int], target: int) -> int:
        n = len(nums)
        pre = [0] * (n * 2 + 1)
        pre[n] = 1
        cnt = n
        ans = presum = 0
        for i in range(n):
            if nums[i] == target:
                presum += pre[cnt]
                cnt += 1
                pre[cnt] += 1
            else:
                cnt -= 1
                presum -= pre[cnt]
                pre[cnt] += 1
            ans += presum
        return ans
```
