# 101109. K 个元素的最大总和
## 关键点
1. 无。
## 解法
贪心，让数组按照由大到小排序，然后枚举；
如果mul是大于1的，那总值就+x*mul
如果mul是小于1的，那总值就+x
## 优化
当mul小于1时，直接将当前值到K值全部加到总值。
#### 代码(184ms)
```python
class Solution:
    def maxSum(self, nums: list[int], k: int, mul: int) -> int:
        ans = 0
        nums.sort(reverse=True)
        n = len(nums)        
        for i in range(min(k,n)):
            if mul<1:
                ans += sum(nums[i:min(k,n)])
                break
            else:
                ans += mul*nums[i]
                mul -= 1
        return ans
```
