# 3633. 最早完成陆地和水上游乐设施的时间 I
## 关键点
1. 无
## 解法
### 1.陆地和水上各玩一样，那就放两种情况，先玩水上，或者先玩陆地，都要找时间结束最早的来玩，这样才能玩下一场结束可能才会更好
#### 时间复杂度
```math
O(n+m)
```
#### 空间复杂度
```math
O(n)
```
#### 代码(7ms)
```python
class Solution:
    def earliestFinishTime(self, landStartTime: List[int], landDuration: List[int], waterStartTime: List[int], waterDuration: List[int]) -> int:
        #假设先玩陆地项目
        min_finish = min(start + duration for start,duration in zip(landStartTime,landDuration))
        land_min = min(max(start,min_finish) + duration for start,duration in zip(waterStartTime,waterDuration))
        #假设先玩水上项目
        min_finish = min(start + duration for start,duration in zip(waterStartTime,waterDuration))
        water_min = min(max(start,min_finish) + duration for start,duration in zip(landStartTime,landDuration))
        
        return min(land_min,water_min)
```
