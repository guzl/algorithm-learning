# 1344. 时钟指针的夹角

## 关键点
1. 时针和分针的较小的夹角。
2.分针的相对于0点位置的偏转角度可以直接算出，但时针的相对于0点的偏转角度要使用小时数和分钟数两个值来确定。
## 解法
### 直接计算
直接计算时针和分针相对于0点的偏转角度，公式如下：
angle_minutes = 360/60 * minutes
angle_hour = 360/12 * (hour%12) + 30/60 * minutes
然后再算分针和小时指针的较小夹角的值
min(abs(angle_minutes - angle_hour),360-abs(angle_minutes - angle_hour))
#### 时间复杂度
```math
O(1)
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def angleClock(self, hour: int, minutes: int) -> float:
        a_minutes = 360/60 * minutes
        a_hour = 360/12 * (hour%12) + 0.5*minutes
        ret = abs(a_minutes-a_hour)
        return min(ret,360-ret)
```
