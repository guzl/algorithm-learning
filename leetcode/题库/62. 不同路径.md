# 62. 不同路径
无。
## 关键点
1.当n或者m为1时，路径的条数只有一条。所以在这个二维数字的第一行和第一列的值为1，
## 解法
### 动态规划
因为机器人只能向下或者向右走，那么在(m,n)这个位置的路径条数就是(m-1,n)的路径条数+(m,n-1)的路径条数。
ret_value[i][j] = ret_value[i-1][j] + ret_value[i][j-1]
- 时间复杂度
```math
O(n^2)
```
- 空间复杂度
```math
O(m*n)
```
```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[1]*n for _ in range(m)]
        for i in range(1,m):
            for j in range(1,n):
                dp[i][j] = dp[i-1][j] + dp[i][j-1]
        return dp[m-1][n-1]
```
