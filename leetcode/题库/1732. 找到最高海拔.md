# 1732. 找到最高海拔

## 关键点
1. 无
## 解法
### 1.遍历
遍历高度差，算出当前高度，并比较更新保存的最高高度，返回最高高度。
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def largestAltitude(self, gain: List[int]) -> int:
        ret = 0
        maxret = 0
        for i in gain:
            ret = ret + i
            if ret > maxret:
                maxret = ret
        return maxret
```
