# 11. 盛最多水的容器
## 关键点
1. 类似于寻找子数组的问题，双循环$O(n^2)$的方式要超时。
## 解法
1. 双指针l,r;从两端向中间启动。
2. 移动条件：左右两个板子，找到短的向内移动
3. 停止条件：$l>=r$是停止 
4. 每一步移动都比较最大值，保存最大值
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(1)
```
#### 代码(59ms)
```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        n = len(height)
        l = 0
        r = n-1
        max_ret = 0
        while l<r:
            max_ret = max(max_ret,(r-l)*min(height[l],height[r]))
            if height[l] < height[r]:
                l+=1
            else:
                r-=1
        return max_ret
```
