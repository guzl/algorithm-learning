# 1833. 雪糕的最大数量
## 关键点
1. 无
## 解法
### 1.排序+贪心
#### 时间复杂度
```math
O(nlog(n))
```
#### 空间复杂度
```math
O(1)
```
#### 代码(87ms)
```python
class Solution:
    def maxIceCream(self, costs: List[int], coins: int) -> int:
        costs.sort()
        ret = 0
        for i in costs:
            if coins >= i:
                ret += 1
                coins -= i
        return ret
```
### 1.hash+贪心
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(m)
```
####  代码(43ms)
```python
class Solution:
    def maxIceCream(self, costs: List[int], coins: int) -> int:
        ret = 0
        max_cost = max(costs)+1
        cost_dict = [0]*max_cost
        for i in costs:
            cost_dict[i] +=1
        for i in range(max_cost):
            while coins >= i and cost_dict[i]>0:
                coins -= i
                ret += 1
                cost_dict[i] -=1
            if coins < i:
                break
        return ret
```
