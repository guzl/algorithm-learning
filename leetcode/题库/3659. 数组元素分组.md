# 3659. 数组元素分组

## 关键点
## 解法
### 1.遍历求和
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def checkGoodInteger(self, n: int) -> bool:
        s = str(n)
        ds = 0;
        ss = 0;
        for d in s:
            ds += int(d)
            ss += int(d)*int(d)
            if(ss - ds)>=50:
                return True
        return False
```
