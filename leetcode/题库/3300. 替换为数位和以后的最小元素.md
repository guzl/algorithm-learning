# 3300. 替换为数位和以后的最小元素
## 关键点
1. 无
## 解法
### 1.遍历，优化
#### 时间复杂度
```math
O(n*log(m))
```
#### 空间复杂度
```math
O(1)
```
#### 代码(0ms)
```python
class Solution:
    def minElement(self, nums: List[int]) -> int:
        min_value = 10**9+1
        for i in range(len(nums)):
            tmp_value = 0
            node_value = nums[i]
            while node_value:
                tmp_value += (node_value%10)
                node_value = node_value//10
                if tmp_value > min_value:
                    break
            min_value = min(tmp_value,min_value)
        return min_value
```
