# 3612. 用特殊操作处理字符串 I

## 关键点
无
## 解法
### 1.遍历，比较，操作
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(1)
```
####  代码
```python
class Solution:
    def processStr(self, s: str) -> str:
        result = ''
        for letter in s:
            if letter == '*':
                result = result[0:-1]
            elif letter == "#":
                result = result + result
            elif letter == "%":
                result = result[::-1]
            else:
                result = result+letter
        return result
```
