# 509. 斐波那契数
无。
## 关键点
1. 无
## 解法
### 动态规划
设置f(0)=0;f(1)=1;然后根据公式遍历即可。
```math
f(n) = f(n-1)+f(n-2)
```
考虑边界问题，我们可以先求出:
```math
f(0)=0,f(1)=1,f(2)=1,f(3)=f(2)+f(1).....,f(n)=f(n-1)+f(n-2)
```
- 时间复杂度
```math
O(n)
```
- 空间复杂度
```math
O(1)
```
```python
class Solution:
    def fib(self, n: int) -> int:
        ret = [0]*(n+2)
        ret[0] = 0
        ret[1] = 1
        for i in range(2,n+1):
            ret[i] = ret[i-1]+ret[i-2]
        return ret[n]
```
