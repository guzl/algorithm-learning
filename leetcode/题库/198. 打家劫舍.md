# 198. 打家劫舍
## 关键点
1. 和登楼梯不一样的地方，就是这次不光可以走1-2步，是可以走很多步
## 解法
### 动态规划
每一步都有选和不选两个状态，
如果选了第n步，它的值就是n步的值+MAX(1到n-1)(不包含n-1)的已经计算出来的最大值；
如果不选第n步，它的值就是MAX(1:n)(不包含n)的已经计算出来的最大值。
最后，选取上述两个情况的最大值即可。
all[0] =0;
如果有1家：all[1] = nums[0]
如果有2家：all[2] = max(nums[0],nums[1])
如果有3家: all[3] = max(nums[0]+nums[2],nums[1])
如果是4家：all[4] = max(nums[3]+max(all[0:3]),max(all[0:4]))
如果是n家：all[n] = max(nums[n-1]+max(all[0:n-1]),max(all[0:n]))
- 时间复杂度
```math
O(n)
```
- 空间复杂度
```math
O(1)
```
```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 1:
            return nums[0]
        if n == 2:
            return max(nums[0],nums[1])
        ret = [0]*(n+1)
        ret[1] = nums[0]
        ret[2] = max(nums[0],nums[1])
        for i in range(3,n+1):
            ret[i] = max(nums[i-1]+max(ret[:i-1]),max(ret[:i]))
        return ret[n]
```
