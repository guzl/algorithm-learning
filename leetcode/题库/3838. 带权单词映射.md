# 3838. 带权单词映射
## 关键点
1. 无
## 解法
### 1.跟着题意走就好了
#### 时间复杂度
```math
O(n)
```
#### 空间复杂度
```math
O(1)
```
#### 代码(6150ms)
```python
class Solution:
    def mapWordWeights(self, words: List[str], weights: List[int]) -> str:
        letters = "abcdefghijklmnopqrstuvwxyz"
        ldict = dict()
        for i in range(26):
            ldict[letters[i]] = weights[i]
        ret_str = ""
        for w in words:
            l_sum = 0
            for s in w:
                l_sum += ldict[s]
            ret_str += letters[-l_sum%26-1]
        return ret_str
```
