# 3976. 乘以系数后最大子数组和
## 关键点
1. 无。
## 解法
排序，合并忙碌时间，在忙碌时间集合中排除空闲时间段。
## 优化
无
#### 代码(135ms)
```python
class Solution:
    def filterOccupiedIntervals(self, occupiedIntervals: List[List[int]], freeStart: int, freeEnd: int) -> List[List[int]]:
        #排序
        #occupiedIntervals.sort()这个排序比下面的排序慢了130ms
        occupiedIntervals = sorted(occupiedIntervals, key=lambda x: x[0])
        merged = [occupiedIntervals[0]]
        for s,e in occupiedIntervals[1:]:
            last_s,last_e = merged[-1]
            if s <= last_e + 1:
                new_s = last_s
                new_e = max(last_e,e)
                merged[-1] = [new_s,new_e]
            else:
                merged.append([s,e])
        res = []
        for s,e in merged:
            #在两边，没有重合
            if e < freeStart:
                res.append([s,e])
            elif s > freeEnd:
                res.append([s,e])
            #有重合，分割出来
            else:
                if s <= freeStart - 1:
                    res.append([s,freeStart-1])
                if freeEnd + 1 <= e:
                    res.append([freeEnd+1,e])
        return res
```
