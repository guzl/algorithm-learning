# 740. 删除并获得点数
无。
## 关键点
1. 当一个nums[i]被选中时，删除nums[i]-1和nums[i]+1的元素，所以nums[i]一旦被选中，下次最优的选着是和nums[i]值一样的元素，因为只用第一次选中的时候付出成本，第二次就没有任何成本了。所以我们可以整理一下nums，将相同数字进行点数合并。
2.
## 解法
### 动态规划
我们将所有的数字的值放在数列中，然后根据下面的公式进行遍历
    all_value[i] = max(all_value[i-2]+all_value[i],all_value[i-1])
这里的i是nums的某个值，当i被选中时，i-1和i+1不能选，但是i+1是后面的事情了，可以不用管。所有求上面的公式就可以找到最大值。
- 时间复杂度
```math
O(max(n))
```
- 空间复杂度
```math
O(max(n))
```
```python
class Solution:
    def deleteAndEarn(self, nums: List[int]) -> int:
        n = max(nums)
        all_value = [0]*(n+1)
        for i in nums:
            all_value[i]+=i
        for i in range(2,n+1):
            all_value[i] = max(all_value[i-2]+all_value[i],all_value[i-1])
        return all_value[n]
```
